# AI Automation Portfolio 

This repository showcases production-ready AI automation blueprints built on n8n, leveraging advanced LLM routing and logic models.

---

## 1. AI Customer Sentiment Router 
* **Trigger:** Listens for incoming customer chat messages.
* **AI Processing:** Uses Google Gemini to instantly analyze message sentiment.
* **Logic:** Detects if a customer is frustrated or happy and handles the response path accordingly.
* **Business Value:** Prevents lost sales by instantly flagging angry clients so customer service teams can reply in seconds.

---

## 2. AI Lead Qualification Router 
* **Trigger:** Captures new inbound leads instantly via webhook from forms or landing pages.
* **AI Processing:** Analyzes intent, company size, and budget using Google Gemini, formatting the decision into structured data.
* **Logic:** Evaluates the generated lead score; if high-value, it routes immediately to high-priority workflows.
* **Business Value:** Shrinks lead response times down to minutes for key targets, dramatically boosting conversion rates.
