# AI Customer Sentiment Router 🤖

This is a functional AI automation blueprint built on a self-hosted instance of n8n.

What it does:**
Trigger: Listens for incoming customer chat messages.
AI Processing: Uses Google Gemini to instantly analyze the message sentiment.
Logic: Detects if a customer is frustrated or happy and handles the response path accordingly.
Business Value: Prevents lost sales by instantly flagging angry clients so customer service teams can reply in seconds.
